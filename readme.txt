# requirements:

node ^20.14.0 is required

# description:

*TodoItem* is model used in examples
*TodoItemsRepository* is a class that holds repository responsibilities
*TodoItemsLogic is a class that holds business logic responsibilities

*TodoItemsManage* is initial class example that mixes different responsibilities
*TodoItemsManagerWithInjection* is a class example that extracts repository responsibilities
*TodoItemsManagerOrchestrated* is a class example that orchestrates repository and business logic operations
